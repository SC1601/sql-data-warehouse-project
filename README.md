# **Data Warehouse and Analytics Project**

<sub>Welcome to the Data Warehouse and Analytics Project repository!🚀</sub>

<sub>This project demonstrates a comprehensive data warehousing and analytics solution, from building a data warehouse
to generating actionable insights. Designed as a portfolio project, it highlights industry best practices in data engineering and analytics.</sub>

## **🚀 Project Requirements**

### **Building the Data Warehouse (Data Engineering)**

<sub>**Objective**</sub>

<sub>Develop a modern data warehouse using SQL Server to consolidate sales data, enabling analytical reporting and informed decision-making.</sub>

<sub>**Specifications**</sub>

<sub>• **Data Sources:** Import data from two source systems (ERP and CRM) provided as CSV files.</sub><br>
<sub>• **Data Quality:** Cleanse and resolve data quality issues prior to analysis.</sub><br>
<sub>• **Integration:** Combine both sources into a single, user-friendly data model designed for analytical queries.</sub><br>
<sub>• **Scope:** Focus on the latest dataset only, historization of data is not required.</sub><br>
<sub>• **Documentation:** Provide clear documentation of the data model to support both business stakeholders and analytics teams.</sub><br>

***

### **BI: Analytics & Reporting (Data Analytics)**

<sub>**Objective**</sub>

<sub>Develop SQL-based analytics to deliver detailed insights into:</sub>

<sub>• **Customer Behavior**</sub><br>
<sub>• **Product Performance**</sub><br>
<sub>• **Sales Trends**</sub><br>

<sub>These insights empower stakeholders with key business metrics, enabling strategic decision-making.</sub>
